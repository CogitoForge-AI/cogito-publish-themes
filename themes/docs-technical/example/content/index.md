---
**Widgetizer** turns loosely-described widgets into validated, production-ready
components. This demo handbook shows what the `docs-technical` theme looks like
in practice: a top navbar, a left navigation sidebar, an "on this page" table of
contents, breadcrumbs, and prev/next links.

## Start here

New to Widgetizer? Read [[Getting started]] to install the toolkit and shape
your first widget, then skim the [[Configuration]] reference to tune it.

## What's inside

- **Guide** -- task-oriented walkthroughs that take you from zero to a working
  setup.
- **Reference** -- the precise, lookup-friendly description of every command
  and option.

## Why a handbook

Good documentation separates *learning* from *looking up*. The guide teaches;
the reference answers. This split keeps each page short and lets the sidebar
double as a map of the whole product.
