**Blog Technical** is built for posts that carry weight: tutorials, deep dives,
and anything with headings and code. Every post gets a collapsible table of
contents, reading time, tag chips, breadcrumbs, prev/next links, and copy
buttons on code blocks.

It suits an engineering blog or a changelog-style site. Like every gallery
theme, you adopt it by vendoring the directory and setting `Config.layout`; the
`blog` preset handles the rest.
